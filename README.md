# homebrew-tap

Homebrew tap for [josepmedialdea](https://github.com/josepmedialdea) projects.

## Usage

```sh
brew tap josepmedialdea/tap
brew install backdrop
```

Or install directly without tapping first:

```sh
brew install josepmedialdea/tap/backdrop
```

> **Note:** If a formula name clashes with one from another tap or homebrew-core (e.g. `backdrop`), use the fully qualified name `josepmedialdea/tap/<formula>` to avoid ambiguity.

## Formulas

| Formula | Description |
|---------|-------------|
| [backdrop](https://github.com/josepmedialdea/backdrop) | CLI tool that fills transparent image backgrounds with a solid color |
